
== WORK IN PROGRESS

Example web application inspired by Digg where a user can create an account, refer to articles online and read what others are posting as well as commenting.

Displays the use of:

* Ember and jQuery for the front-end

* Skeleton CSS framework

* RSpec for tests

* Devise for user and sessions management

* SQL and Mongo databases interacting one with another

* Nokogiri for web scrapping
